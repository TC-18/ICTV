# Interactive Curvature Tensor Visualization

### Build & Run

```sh
cd src/GKIT
premake4 gmake
make vizo
./vizo <data> 1 <size>
```
* \<data\> - any other .raw file
* \<size\> - object's bounding box.

#### Example

```sh
./vizo ../../data/raw/octa-flower-130_border.raw 1 130
```

#### Dependencies

* SDL2
* SDL2_image
* SDL2_ttf
* openexr
* glew
